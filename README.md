# Apple Health Step Analysis

## Описание проекта

Данный проект анализирует данные о шагах, экспортированные из приложения Apple Health на iPhone.

Проект:
- загружает XML-файл из Apple Health
- преобразует данные в CSV
- анализирует физическую активность пользователя
- выводит статистику и аналитические выводы

В проекте используются:
- pandas
- NumPy
- генераторы
- декораторы
- pytest
- CLI-интерфейс

## Функциональность проекта

Проект поддерживает:

- рекордные даты по количеству шагов
- антирекордные даты
- среднее количество шагов
- анализ активности по дням недели
- анализ активности по месяцам
- анализ активности по годам
- анализ выполнения цели по шагам
- анализ стабильности активности
- поиск серий активных дней

## Используемые технологии

### pandas
Используется для:
- обработки таблиц

- анализа дат

Используется для:

Используются для:

## Подготовка данных

Экспортируйте данные из приложения Apple Health и поместите файл `export.xml` в папку проекта.

После этого запустите:

```bash
python data_loader.py
```

Будет создан файл:

```text
daily_steps.csv
```

---

## Запуск проекта

Запуск основного приложения:

```bash
python main.py
```

---

## Запуск тестов

```bash
python -m pytest
```

## Примеры аналитических выводов

- В какие дни недели пользователь наиболее активен
- В каком месяце наблюдалась максимальная активность
- Насколько стабильна физическая активность
- Как часто выполнялась цель в 10000 шагов
- Какие периоды были наиболее активными
